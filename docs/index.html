ภาพรวม

เรามาดูกันว่าเปอร์เซ็นต์ที่ชาวต่างชาติเข้ามาที่ประเทศไทยในแต่ละเดือนของปี2017นั้นเป็นยังไง จากกราฟเราจะเห็นได้ว่าในเดือนมกราคมมีนักท่องเที่ยวเข้ามามากที่สุด เนื่องมาจากเป็นช่วงปีใหม่ของแต่ละประเทศเเละยังมีอากาศที่ค่อนข้างเย็นสบายจึงมีเปอร็เซ็นต์ที่นักท่องเที่ยวเข้ามามากที่สุด

และจากกราฟจะเห็นว่า ในเดือนเมษายนมีเปอร์เซ็นต์นักท่องเที่ยวเข้ามาน้อย เนื่องมาจากช่วงเมษาของประเทศไทยมีอากาศค่อนข้างร้อนมากนักท่องเที่ยวที่มาจากต่างชาติอาจไม่ค่อยชอบนักเพราะชาวต่างชาติอาจชินกับอากาศที่สบายๆของประเทศตัวเองแต่ถ้ามาประเทศไทยในช่วงที่ร้อนมากอาจไม่สบายตัวจึงนิยมมาช่วงหน้าหนาวหรือช่วงที่มีอากาศสบายๆอย่างเดือนมกราคมมากกว่า

ดังนั้น เราก็จะมาวิเคราะห์กันต่อจากกราฟว่าในแต่ละฤดูที่นักท่องเที่ยวต่างชาติในแต่ละทวีปนั้นมาเที่ยวประเทศไทยมีกี่คนทวีปไหนมามากที่สุดและน้อยที่สุดฤดูไหน ไปดูกันเลย

ฤดูหนาว

มาดูข้อมูลที่ละเอียดลึกลงมากัน ตอนนี้จะสามารถดูได้ว่าในแต่ละปีได้มีนักท่องเที่ยวต่างชาติเข้ามาตั้งแต่ปี 2013 จนถึงปี 2018

จากกราฟจะสังเกตได้ว่า ในช่วงฤดูหนาวเดือนตุลาคมถึงต้นเดือนกุมภาพันธ์ มีนักท่องเที่ยวจากทวีปเอเชียเข้ามาเที่ยวในไทยมากที่สุดรองลงมาเป็นทวีปยุโรปส่วนทวีปอื่นเข้ามาเป็นสัดส่วนน้อยจากกราฟจึงเห็นสัดส่วนที่ไม่ค่อยชัดนักและจำนวนที่เข้ามาแต่ละปีค่อนข้างคงที่ อาจเป็นเพราะประเทศไทยยังไม่เป็นที่รู้จักในทวีปเหล่านี้มากนัก

ส่วนในทวีปเอเชียจากกราฟจะสังเกตได้ว่าเป็นทวีปที่เข้ามาไทยเป็นอันดับต้นๆโดยเฉพาะประเทศจีน, มาเลเซีย, เกาหลีใต้และญี่ปุ่นโดยในปี2017 จะเห็นว่านักท่องเที่ยวทวีปเอเชียเข้ามาเที่ยวสูงที่สดเนื่องมาจากนโยบายส่งเสริมจากทางรัฐที่ต้องการกระตุ้นเศรษฐกิจจึงได้ให้มีการต่ออายุมาตรการยกเว้นค่าธรรมเนียมวีซ่า และมาตรการปรับลดค่าธรรมเนียมวีซ่าที่ด่านตรวจคนเข้าเมือง และ ประเทศไทยก็อยู่ในทวีปเอเชียด้วยเช่นกัน ใกล้รองลงมาเป็นทวีปยุโรปที่อยู่ติดทวีปเอเชียพอดี

ส่วนในทวีปอื่นๆ ที่มีปริมาณเข้ามาท่องเที่ยวภายในประเทศไทยน้อยเป็นเพราะว่าอยู่ห่างไกลจากประเทศไทยมากทำให้ค่าใช้จ่ายในการเดินทางราคาแพงและประเทศไทยยังไม่เป็นที่รู้จักดีพอในทวีปอื่นๆ

รายได้

มาดูข้อมูลที่ละเอียดลเกี่ยวกับรายได้กัน ตอนนี้จะสามารถดูได้ว่าในแต่ละปีได้รับรายได้จากนักท่องเที่ยวต่างชาติตั้งแต่ปี 2013 จนถึงปี 2018

วิเคราะห์ข้อมูล

จากการวิเคราะห์ข้อมูลจากกราฟแสดงรายได้จากนักท่องเที่ยวต่าวชาติเข้ามาในประเทศไทย ตั้งแต่ปี 2013-2018 สังเกตุได้ว่าจำนวนตั้งแต่ปี 2013 จนถึงปี 2015 รายได้ที่ได้รับจากนักท่องเที่ยวต่างชาติลดลงทีละน้อย เพราะอาจมีปัจจัยและสถานการณ์ต่างๆในประเทศไทยเราเป็นผล ทำให้นักท่องเที่ยวต่างชาติจากแต่ละทวีปต่างๆลดน้อยลง โดยเฉพาะทวีปเอเชียตะวันออกและรองลองมาก็ทวีปยุโรป ทุกๆปีรายได้ส่วนใหญ่จะมาจากทั้งสองทวีปนี้ แต่หลังจาก ปี 2015 รายได้ก็เติบโตดีขึ้นเรื่อยๆในแต่ละปี ซึ่งในปี 2018 เมื่อเปรียบกับปี 2017 อาจจะน้อยกว่า แต่รายได้ที่ได้รับนับเพียงแค่เดือนมกราคมถึงเดือนตุลาคมเท่านั้น ถ้าเทียบกันในระยะเวลาที่เท่ากัน รายได้ของปี 2018 เพิ่มมากขึ้นถึง 9.98% เลยทีเดียว ถือว่าเป็นการเติบโตของเศรษฐกิจไทยที่ดีขึ้น

จัดอันดับรายได้

มาดูการจัดอันดับรายได้จากนักท่องเที่ยวต่างชาติเดินทางเข้าไทยมากที่สุด ทั้ง 7 ทวีป ตั้งแต่ปี 2013 จนถึงปี 2018 ดังนี้

อับดับ 1 คือ ทวีปเอเชียตะวันออก(EastAsia) เป็นจำนวน 4,441,489 ล้านบาท รองลงมา ทวีปยุโรป(Europe) 2,477,618 ล้านบาท, ทวีปอเมริกา(America) 528,438 ล้านบาท, ทวีปโอเชียเนีย(Oceania) 408,614 ล้านบาท, ทวีปเอเชียใต้(SouthAsia) 355,099 ล้านบาท, ทวีปตะวันออกกลาง(MiddleEast) 316,805 ล้านบาท, ทวีปแอฟริกา(Africa) 66,421 ล้านบาท ตามลำดับ
